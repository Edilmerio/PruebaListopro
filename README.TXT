1- Instalar python version 3.6 o superior
2- Clonar el proyecto (git clone https://github.com/Edilmerio/PruebaListopro.git)
3- Crear entorno virtul (venv)
4- Activar venv (pythom -m venv venv)
5- Ubicarse en el directorio PruebaListopro/
6- Run pip install -r requirements.txt
7- Run yarn install --flat
8- Run python manage.py makemigrations
9- Run python manage.py migrate
10-Run python manage.py collectstatic for production environment
11-Crear y cargar en el fichero PruebaListopro/events.json el JSON de los eventos de google calendar
